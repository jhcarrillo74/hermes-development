# Hermes Development Documentation

Repositorio de documentación y workflows para desarrollo de Hermes Agent.

## 📁 Contenido

- `hermes-master.md` - Documentación principal y estándares de desarrollo
- `workflow-existing-feature.md` - Flujo para modificar features existentes
- `workflow-new-feature.md` - Flujo para crear nuevas features

## 🚀 Quick Start

1. **Revisar estándares** en `hermes-master.md` antes de desarrollar
2. **Seguir workflow** apropiado según el tipo de cambio
3. **Mantener documentación** actualizada con los cambios

## 📋 Workflows

- **Nueva feature**: Usar `workflow-new-feature.md`
- **Feature existente**: Usar `workflow-existing-feature.md`
